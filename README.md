# the_plainly_stated
React Website - the start of THE streetwear brand.

## Steps
Make sure NPM is installed and up to date.
`` Brew NPM install ``
Add npm to project
`` npm init ``
Add Webpack - dev dependency only
`` npm i webpack webpack-cli -D ``
Add ReaCT - dev & prod dependency
`` npm i react react-dom -S ``
